Introduction
------------
Module provides dynamic page background functionality.

Main features are:
 - backgrounds rotation with lightweight js library "superBgImage"
 - different background sets for selected pages. (Configured as usual
 block's visibility rules)

Installation
-------------
Install this module as any other module

Generic workflow
-------------------------------
1. After module installed you can navigate to "admin/content/background_set"
and click on "Add Background set"
2. Enter Background set title and upload images for page background. Created
set should be available at Background changer block configuration form.
3. Place Background changer block on a page region.
4. Select Background set by title. You could also specify container element
id and slideshow interval or just go with defaults.
